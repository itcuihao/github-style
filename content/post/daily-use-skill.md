## 快捷方式

使用`mac`终端，在当前目录下，想打开`finder`，可在终端中输入：

```
➜  / open .
```

使用`mac`终端，在`finder`中，想快速跳转到某一个目录，可使用快捷键：

```
Command + Shift + G
```

编辑Vim文件后保存发现权限不足：

```
命令模式: w !sudo tee %
```

保存Vim文件：

```
一般是 :wq
不过可以 :x
```
